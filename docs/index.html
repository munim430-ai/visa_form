What it does

Everything you need to process student visa applications

📊
Excel Input
Fill one row per student in a structured Excel template. Download the template directly from the app.
📄
Visa Application DOCX
Generates a fully filled Korean D-4-1 visa application form per student — ready to print and submit.
✍️
NOC Certificate
Auto-generates a Parents' No Objection Certificate alongside the visa form — two documents per student.
🖼️
Photo Embedding
Automatically embeds the student's photo into the visa form. Just name the photo file and drop it in the photos folder.
☑️
Smart Checkboxes
Education level, gender, and marital status checkboxes are auto-filled based on Excel data. No manual editing.
⚡
Batch Processing
Process dozens of students in one click. Each student gets their own named DOCX files — no mixing up forms.

Simple workflow

From Excel to ready-to-submit forms in minutes

1

Download & Run

Download the portable EXE — no installation needed. Double-click to open. Log in with your agency password.

Windows 10/11 · 64-bit
2

Fill the Excel Template

Download the input template from inside the app. Fill one row per student — name, passport, university, sponsor, education, and more.

40+ fields · dropdown validation
3

Upload & Generate

Upload the filled Excel. Click Generate. The app creates one Visa Application DOCX and one NOC DOCX per student — automatically.

Batch · instant
4

Download Each File

Every student gets two clearly-named download buttons — one for the visa form and one for the NOC. Print and submit.

STU001_NAME.docx · NOC001_NAME.docx
